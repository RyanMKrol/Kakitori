# T036 — Home deck list: deck cards, mastery %, status chips, today banner

## 2026-07-18 — Implementation complete

### What was done

Implemented the Home screen deck list UI components:

1. **TodayBannerView.swift** — Dark banner showing total due cards across all decks
   - Displays "N characters to write · across M scripts"
   - Hidden when N = 0 (no cards due)
   - Iterates through all decks/sections/notes to calculate totals
   - Filters by schedule state and due date using injected `now` parameter

2. **DeckCardView.swift** — Individual deck card component in 2-up grid
   - Japanese title (ひらがな) at 32pt using Hiragino Mincho
   - English title + card count subtitle
   - Mastery % badge (right-aligned, accent background) = mature cards / total (mature = intervalDays >= 21)
   - Three status chips: "N new" / "N learning" / "N due"
   - "Study →" label with accent color
   - Full card is tappable with `onStudy(Deck)` closure
   - Accessibility identifier: `deck-row-<sourceDeckName.lowercased()>`

3. **HomeView.swift** updates
   - Added `TodayBannerView` below stats row when decks exist
   - Replaced List with `LazyVGrid(columns: 2-up)` for deck cards
   - Enhanced empty state with "Import deck" button (identifier: `import-button-empty`)
   - Uses `AppClock.system.now()` for injected time source
