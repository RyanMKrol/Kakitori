# T044 — Recall mode (思い出し)

## 2026-07-18

### Implementation

Added the Recall (思い出し) practice mode prompt and canvas for session practice. The mode presents reading (romaji/kana) with optional English meaning in curly quotes, and empty guide boxes for writing from memory.

**Changes made:**

1. **PromptPaneView.swift**: Added `recallModePrompt` that:
   - Shows "RECALL MODE" caption in accent small caps
   - Shows "Reading" label (secondary color)
   - Displays pronunciation (~40pt Japanese display font)
   - Shows English meaning in curly quotes when present (using Unicode U+201C/U+201D)
   - Shows sub-line: "Write the word from memory."
   - Respects `showRomaji` setting: when false AND English is present, hides pronunciation headline; when false but no English, shows pronunciation anyway (kana decks must never be blank)

2. **CanvasPaneView.swift**: Added recall canvas hint:
   - Hint text: "Write the word from memory" (matches prototype)
   - Guide boxes remain empty (no faded trace layer)

3. **Tests**: Added comprehensive render tests:
   - `testRecallModePromptWithReadingAndEnglish` — shows reading + English + instruction
   - `testRecallModePromptWithoutRomaji` — shows only English when romaji disabled
   - `testRecallModePromptNoEnglish` — shows reading even when romaji disabled (kana deck safety)
   - `testRecallModeAnswerAfterReveal` — verifies answer block displays correctly
   - `testRecallModeCanvasHidesGuideBoxes` — verifies canvas hint and empty guide boxes
