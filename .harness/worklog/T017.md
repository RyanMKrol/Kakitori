# T017 — AnkiCollection: read models/decks/notes from collection.anki2

## Completed 2026-07-17

### What was built

Created `Sources/Importer/AnkiCollection.swift` with:
- **Value types:** `AnkiModel`, `AnkiDeckInfo`, `AnkiNote` (all Sendable)
- **Error enum:** `AnkiCollectionError` with cases: `cannotOpen`, `malformedCollection`, `noAnkiBuilderModel`, `zeroNotes`
- **Main struct:** `AnkiCollection` with read-only SQLite3 initializer

### Implementation details

The `init(databaseURL:)` initializer:
1. Opens database read-only via `sqlite3_open_v2(SQLITE_OPEN_READONLY)`
2. Reads `col.models` and `col.decks` as JSON blobs, parsed with `JSONSerialization`
3. Extracts field names from models sorted by `ord` to preserve order
4. Reads notes from `notes` table, splitting `flds` on `\u{1F}` (unit separator)
5. Maps notes to decks via `cards` table (first card per note wins)
6. Validates: at least one model has `"Target"` field, and notes are non-empty
7. No force-unwraps; all NULL pointers from SQLite throw `.malformedCollection`

### Tests created
