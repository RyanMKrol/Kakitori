### Implementation notes
- Used a private `EOCDData` struct to avoid SwiftLint's large-tuple violation (>2 members)
- Decompression uses `Compression` framework's `compression_decode_buffer()` with `COMPRESSION_ZLIB`
- Private helper struct `ZipEntry` holds parsed central-directory metadata
- Temporary `ZipArchive` instance created during init to access helper methods before `entries` is set
- All reads from `Data` are bounds-checked; out-of-bounds → `.malformed` error
- Path traversal protected: `extractAll()` skips entries containing `..`
- Directory entries skipped in `extractAll()` (names ending in `/`)
