# T011 — Guide box row component (genkō yōshi boxes)

## Completed

**2026-07-17**

### What I built
- Created `Sources/Features/Session/Canvas/GuideBox.swift`: A SwiftUI shape component that renders a square box (1:1 aspect) with:
  - Hairline border (0.5pt) using `KakitoriTheme.boxLine` (8% ink)
  - Dashed centre cross-lines (both horizontal and vertical) through the box center using `StrokeStyle(lineWidth: 0.5, dash: [4, 3])`
  - Paper background fill

- Created `Sources/Features/Session/Canvas/GuideBoxRow.swift`: A layout component that:
  - Takes `units: [SegmentedUnit]` and `maxBoxesPerRow: Int = 6`
  - Wraps boxes into rows with ≤ `maxBoxesPerRow` boxes per row (inline punctuation doesn't count toward limit)
  - Uses `GeometryReader` to measure available width and calculate box size: `min(160, availableWidth / boxesInRow)`
  - Renders `.box(String)` as `GuideBox` with Japanese font
  - Renders `.inline(String)` (punctuation) as plain `Text` in `Hiragino Mincho ProN` font, vertically centered on box midline, NO box
  - Wraps rows and centers them
  - Assigns accessibility identifiers: `"guide-box-N"` for each box, `"guide-box-row"` for container
