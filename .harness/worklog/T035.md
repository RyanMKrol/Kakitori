# T035 — Home stats row: streak, written today, minutes studied

## 2026-07-18

### What was done

Created `StatsRowView.swift` to display three stat cards on the Home screen:
- Day streak (with 🔥 emoji) — computed from the existing `DailyStats.currentStreak()` function over all active days
- Cards written today — fetched from today's `DailyStats` row using the 4 AM-adjusted day
- Minutes studied — computed as `secondsStudied / 60` formatted with trailing "m"

All three cards use:
- White background with hairline borders (`KakitoriTheme.boxLine`)
- Rounded corners (18pt radius)
- Large numerals (28pt semibold)
- Small caps captions (11pt)
- Streak numeral in accent color; others in ink

The row is placed above the deck list in HomeView and displays correctly with both seeded demo data and zero values when no data exists.
