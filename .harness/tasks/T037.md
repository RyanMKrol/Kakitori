# T037 — Deck setup sheet: practice mode picker

## Overview

Build the deck setup sheet: deck header with close, one selectable row per practice mode
(glyph + label + description + selection ring), and the red "Start writing" button.

## Do

- Create `Sources/Features/DeckSetup/DeckSetupSheet.swift` (new folder). Decoupled init so it renders
  without a live `Deck`: `init(jpTitle: String, enTitle: String, dueCount: Int, onStart: @escaping
  (PracticeMode) -> Void, onClose: @escaping () -> Void)`. For the mode type: reuse the practice-mode
  enum if `Sources/Features/Session/SessionViewModel.swift` already defines one; otherwise define
  `enum PracticeMode: String, CaseIterable, Sendable { case trace, listen, translate, recall, mixed }`
  in this file. Presenting the sheet from Home is the later end-to-end task — not here.
- **Header band** (docs/06-ui-design.md §2.2, reference prototype): `KakitoriTheme.ink` fill, paper
  text — jpTitle in `japaneseDisplayFont` ~28pt, secondary line `"\(enTitle) · \(dueCount) cards due"`
  (e.g. "Hiragana · 10 cards due"), circular ✕ close button on the right → `onClose`.
- Small-caps section caption `PRACTICE MODE` (reference prototype).
- **Mode rows** — list ALL five; hiding unavailable modes (docs/02-product-spec.md §2.5) is a later
  availability task, do NOT filter here (say so in a comment). Each row: ~44pt rounded glyph tile
  (Mincho), bold label, one-line faint description, trailing selection ring (hairline circle;
  accent-filled dot when selected); tapping selects. Glyphs per docs/06 §2.2 (書 / 聞 / 訳 / 思 —
  the prototype's 記 for Recall is superseded); labels per docs/02 §2; descriptions from the
  reference prototype's mode list; identifiers from the house list:
  - `書` **Trace** — "Write over a faded guide" — `"mode-trace"`
  - `聞` **Listen & Write** — "Hear it, then write what you heard" — `"mode-listen"`
  - `訳` **Translate & Write** — "See the English, write the Japanese" — `"mode-translate"`
  - `思` **Recall** — "From the reading, write from memory" — `"mode-recall"`
  - `混` **Mixed** — "Rotate through all four modes" — `"mode-mixed"` (Mixed glyph from the prototype)
- Default selection: Trace (the only playable mode until the other mode tasks land).
- Pinned at the bottom: `Start writing` — full-width, `KakitoriTheme.accent` fill, paper text,
  identifier `"start-writing"` → `onStart(selection)` (docs/06 §2.2).
- Render check: `Tests/UnitTests/DeckSetupRenderTests.swift` — construct
  `DeckSetupSheet(jpTitle: "ひらがな", enTitle: "Hiragana", dueCount: 10, ...)` with no-op closures,
  frame ~560×700, `ImageRenderer` scale 2, write `<repoRoot>/screenshots/T037-decksetup.png`
  (path derived from `#filePath`).
