# T044 — Recall mode (思い出し)

## Overview

Add the Recall session prompt: the reading (romaji `pronunciation`), the English meaning in quotes
when present, and "Write the word from memory." over empty guide boxes (docs/02-product-spec.md §2.4).

## Do

1. In the session prompt view in `Sources/Features/Session/` (driven by
   `Sources/Features/Session/SessionViewModel.swift`), add the pre-reveal prompt for
   `PracticeMode.recall` (enum in `Sources/Features/Session/PracticeMode.swift`), reading fields
   from `Sources/Models/Note.swift`:
   - Accent small-caps caption "Recall mode" (same style as the existing Trace caption; prototype).
   - A small secondary-color field label "Reading".
   - The note's `pronunciation ?? ""` as the headline — `KakitoriTheme` Japanese display font,
     ~40pt (docs/06-ui-design.md §2.3 prompt pane). Never force-unwrap.
   - When `english` is non-empty: a quieter line with the English in curly quotes, exactly
     `“` + english + `”` (docs/02 §2.4: "when present, the English meaning in quotes").
   - Sub-line: "Write the word from memory." (docs/02 §2.4 and the prototype, with the period).
2. Romaji setting: read `(UserDefaults.standard.object(forKey: "showRomaji") as? Bool) ?? true`
   (default on, docs/02 §5). When false AND `english` is non-empty, hide the pronunciation headline
   (the quoted English carries the prompt). When false but `english` is nil or empty, show the
   pronunciation anyway — a Recall prompt must never be blank (kana decks prompt with romaji,
   docs/02 §2.5).
3. Canvas: in `.recall` the guide boxes are EMPTY (trace layer renders only for `.trace`). Canvas
   hint line for recall: "Write the word from memory" (prototype box hint, no period).
4. iPhone/portrait: same content in the compact prompt band above the canvas, like the existing
   Trace band (docs/06 §2.3).

## Done when

- `./build_run.sh Kakitori-Sim -seedDemoData YES`, then drive the UI into a session with the
  "Recall" mode selected: the screenshot shows the RECALL MODE caption, the "Reading" label, the
  romaji reading, the quoted English line (for a seeded note that has English), the sub-line
  "Write the word from memory.", and guide boxes with NO faded guide glyphs.
- The Japanese target is nowhere on screen before Show answer.
