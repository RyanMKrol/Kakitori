# T005 — SwiftData model: Note + Script classification

## Overview

Create the `Note` model — the imported card content — plus the `Script` enum that classifies what
kind of Japanese text the note's target is, per docs/05-architecture.md §3.

## Do

1. Create `Sources/Models/Note.swift` (`import SwiftData`, `import Foundation`):
   - `enum Script: String, Codable, Sendable { case hiragana, katakana, kanji, mixed }`
     (docs/05 §3: "hiragana | katakana | kanji | mixed").
   - `@Model final class Note` with exactly these stored fields (docs/05-architecture.md §3):
     - `var id: UUID`
     - `var target: String` — text to write
     - `var pronunciation: String?`
     - `var english: String?`
     - `var category: String?`
     - `var hint: String?`
     - `var audioFilename: String?` — relative to `Media/<deckUUID>/`
     - `var script: Script`
     - `var units: [String]` — segmented writeable characters
     - `var isDeleted: Bool` — soft delete on re-import
     - `var section: Section?` — plain `var` (inverse declared on the Section side, below)
     - Do NOT declare the `schedule` property in this task: docs/05 lists `var schedule: CardSchedule?`
       on Note, but `CardSchedule` does not exist yet — the next model task creates
       `Sources/Models/CardSchedule.swift` and adds `schedule` to Note then.
   - `init` with defaults: `id: UUID = UUID()`, optionals `= nil`, `units: [String] = []`,
     `isDeleted: Bool = false`. Required params: `target`, `script`.
2. Edit `Sources/Models/Deck.swift` (created earlier; contains `Deck` and `Section`): add to
   `Section`: `@Relationship(deleteRule: .cascade, inverse: \Note.section) var notes: [Note]`,
   defaulting to `[]` in Section's `init`.

No force-unwraps or `try!`; no `Date()` reads.

## Done when

- `Tests/UnitTests/NoteModelTests.swift` (XCTest) uses an in-memory container
  (`ModelConfiguration(isStoredInMemoryOnly: true)` over `Deck.self, Section.self, Note.self`) —
  never the real store — and asserts:
  - A `Note(target: "ありがとう", pronunciation: "arigatou", english: "thank you", script: .hiragana,
    units: ["あ","り","が","と","う"])` inserts and fetches back with every field intact, including
    the `units` array order and `isDeleted == false`.
  - Appending the note to a `Section.notes` round-trips: refetched `section.notes.count == 1` and
    `note.section === section` (relationship integrity).
  - `Script(rawValue: "kanji") == .kanji` and `Script.mixed.rawValue == "mixed"`.
