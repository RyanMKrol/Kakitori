## Overview
Trace mode is currently auto-playing the card's pronunciation as the card loads. That is wrong: in Trace (the default mode) audio must play ONLY when the user explicitly taps the speaker/replay button, never automatically on card entry or on advance. An earlier change deliberately broadened autoplay from Listen-only to also include Trace; this reverses that so only Listen auto-plays. Translate already correctly does not autoplay; this task must not change Listen or Translate behavior.
