# T043 — Translate & Write mode (翻訳)

## Overview

Add the Translate & Write session prompt: the English meaning with "Write this in Japanese.", empty
guide boxes (docs/02-product-spec.md §2.3). This task implements the prompt only — hiding the mode
for decks/cards without an English gloss is enforced by the mode-availability task
(`.harness/tasks/T045.md`), not here.

## Do

1. In the session prompt view in `Sources/Features/Session/` (driven by
   `Sources/Features/Session/SessionViewModel.swift`), add the pre-reveal prompt for
   `PracticeMode.translate` (enum in `Sources/Features/Session/PracticeMode.swift`):
   - Accent small-caps caption "Translate & write" (same style as the existing Trace caption).
   - A small secondary-color field label "English" (prototype prompt pane).
   - The note's `english` (from `Sources/Models/Note.swift`) as the headline — bold, ~34pt, wrapping,
     `KakitoriTheme.ink`. Render `english ?? ""` via nil-coalescing; never force-unwrap. A nil or
     empty `english` may render blank in this task — availability filtering lands in
     `.harness/tasks/T045.md`.
   - Sub-line below: "Write this in Japanese." (docs/02 §2.3 and the prototype, character for
     character, including the trailing period).
   - No target, reading, or kana text before reveal — the answer block after Show answer is the
     existing standard one, unchanged.
2. Canvas: in `.translate` the guide boxes are EMPTY (no faded trace glyphs — the trace layer renders
   only for `.trace`). Canvas hint line for translate: "Write the Japanese here" (prototype).
3. iPhone/portrait: the same content collapses into the prompt band above the canvas (caption +
   headline + one sub-line), matching how the Trace band already behaves (docs/06-ui-design.md §2.3).

## Done when

- `./build_run.sh Kakitori-Sim -seedDemoData YES`, then drive the UI into a session with the
  "Translate & write" mode selected (seeded demo data includes notes with English): the screenshot
  shows the TRANSLATE & WRITE caption, the "English" field label, the English meaning as the
  headline, the line "Write this in Japanese.", and guide boxes with NO faded guide glyphs.
- The Japanese target is nowhere on screen before Show answer.
