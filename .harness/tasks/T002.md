# T002 — KakitoriTheme design tokens (palette, type, shape)

## Overview

Create the single source of truth for Kakitori's visual language — the "quiet stationery" palette,
Japanese display font helpers, corner radii, and count-chip styles — so every later UI task consumes
named tokens instead of re-deriving hex values.

## Do

Create `Sources/Support/Theme.swift` containing one public-ish enum `KakitoriTheme` (caseless
namespace, no instances) with these static tokens:

- **Palette** (docs/06-ui-design.md §1 Palette — light mode values, quoted exactly):
  - `paper` = warm off-white `#FAF8F4` (app background)
  - `ink` = near-black `#1A1A1A` (text, user strokes)
  - `inkFaint` = `ink` at 12% opacity (guide glyphs in Trace mode)
  - `accent` = vermillion red `#C43B2E` (hanko red — mode labels, ANSWER caption, primary buttons, streak)
  - `boxLine` = `ink` at 8% opacity (guide box borders + dashed cross lines)
  - Define these as SwiftUI `Color` via a private `Color(hex:)` helper or explicit
    `Color(red:green:blue:)` component math — no force-unwraps.
- **Chip styles** (docs/06-ui-design.md §1 Palette: "neutral fill / red-tint fill / ink fill"):
  - `chipNewBackground` = `ink` at 8% opacity, `chipNewForeground` = `ink`
  - `chipLearnBackground` = `accent` at 12% opacity, `chipLearnForeground` = `accent`
  - `chipDueBackground` = `ink`, `chipDueForeground` = `paper`
- **Shape** (docs/06-ui-design.md §1 Shape & depth: "large radius (16–20pt)"):
  - `radiusLarge: CGFloat = 20` (cards), `radiusMedium: CGFloat = 16` (sheets/chips-container)
- **Typography** (docs/06-ui-design.md §1 Typography — Japanese display is **Hiragino Mincho ProN**):
  - `static func japaneseDisplayFont(size: CGFloat, bold: Bool = false) -> Font` returning
    `Font.custom("HiraMinProN-W6", size: size)` when bold, `Font.custom("HiraMinProN-W3", size: size)`
    otherwise (the PostScript names for Hiragino Mincho ProN).
  - `static func smallCapsLabel(size: CGFloat = 12) -> Font` — system font with generous tracking is
    applied at call sites; here just return `.system(size: size, weight: .semibold)` (UI text stays SF Pro).

Swift 6 strict concurrency: all stored tokens are `let`/computed statics on the caseless enum
(inherently Sendable). No `Date()`, no UIKit imports — SwiftUI only.

## Done when

- `Sources/Support/Theme.swift` compiles with exactly the tokens above; the five palette colors
  match `#FAF8F4`, `#1A1A1A`, 12%-ink, `#C43B2E`, 8%-ink.
- `japaneseDisplayFont(size:bold:)` uses the Hiragino Mincho ProN family (HiraMinProN-W3/W6).
- No other source files are modified; no force-unwraps or `try!` anywhere in the file.
