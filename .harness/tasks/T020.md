# T020 — ApkgImporter: first-import upsert into SwiftData

## Overview

The orchestrating actor of the import pipeline (docs/04-content-and-data.md §2, docs/05 §6):
unzip → read SQLite → map fields → segment targets → insert `Deck`/`Section`/`Note` rows plus one
`CardSchedule` per note in state `.new`. First import only — re-import idempotence is T021.

## Do

The SwiftData models ALREADY exist with exactly the docs/05 §3 fields — read them first:
`Sources/Models/Deck.swift` (`Deck`, `Section`), `Sources/Models/Note.swift` (`Note`, `Script`),
`Sources/Models/CardSchedule.swift` (`CardSchedule`, `CardState`). `TargetSegmenter` ALREADY
exists at `Sources/Importer/TargetSegmenter.swift` (segments a Target into writeable units and
classifies its `Script`) — read it for its exact API. Do NOT edit `Sources/Models/` (out of
scope); only create files under `Sources/Importer/` and `Tests/`.

1. Create `Sources/Importer/NoteIdentity.swift`: `enum NoteIdentity` with
   `static func uuid(forAnkiGUID guid: String) -> UUID` — SHA-256 (`import CryptoKit`) of the
   guid's UTF-8 bytes, first 16 digest bytes packed into a `uuid_t` → `UUID(uuid:)`.
   Deterministic: the same Anki guid ALWAYS yields the same UUID. This derived UUID becomes
   `Note.id`, and is how T021 matches existing notes on re-import (the Anki guid is not stored
   as its own model field).
2. Create `Sources/Importer/ApkgImporter.swift`:
   - `enum ImporterError: Error, Equatable`: `badZip`, `noAnkiBuilderModel`, `zeroNotes`
     (the three validation errors of docs/05 §6).
   - `struct ImportedNote: Sendable`: `ankiGUID: String`, `target: String`, `pronunciation`,
     `english`, `category`, `hint`, `audioFilename` (all `String?`), `sectionName: String?`,
     `units: [String]`, `script: Script`.
   - `struct ParsedDeck: Sendable { let deckName: String; let notes: [ImportedNote] }`.
   - `actor ApkgImporter { init(container: ModelContainer, mediaBaseURL: URL) }` — inject the
     `ModelContainer` (Sendable) and create a fresh `ModelContext(container)` inside the actor;
     never pass a `ModelContext` across the actor boundary (Swift 6 strict concurrency).
     `mediaBaseURL` is stored for T021, which wires `MediaStore` in — this task does NOT copy
     media.
   - `func importDeck(from url: URL) async throws` = `parse(url:)` + `apply(_:)`. Keep both
     `internal` (not private) so tests can drive `apply` with doctored input via
     `@testable import`.
   - `func parse(url: URL) throws -> ParsedDeck`: extract with `ZipArchive` into
     `FileManager.default.temporaryDirectory` + UUID (any `ZipArchiveError` → `.badZip`); open
     `AnkiCollection` on the extracted `collection.anki2`, mapping its `noAnkiBuilderModel` /
     `zeroNotes` to the matching `ImporterError`; pick the first model whose `fieldNames`
     contains `"Target"`; map that model's notes through `NoteFieldMapper` (skip notes of other
     models and notes throwing `emptyTarget`); run `TargetSegmenter` for `units` + `script`.
     Deck naming per docs/04 §1.2: the Kakitori deck name is the top-level part (before the
     first `"::"`) of the Anki deck holding the most notes, ignoring an empty `"Default"` deck;
     each note's `sectionName` is its own Anki deck name with that `"<top>::"` prefix removed
     (`nil` when the note sits directly in the top-level deck).
   - `func apply(_ parsed: ParsedDeck) throws -> UUID`: insert one `Deck` (`id: UUID()`,
     `name: parsed.deckName`, `jpTitle: nil`, `sourceDeckName: parsed.deckName` — the re-import
     matching key — `importedAt: Date()`); one `Section` per distinct non-nil `sectionName`
     (use the existing `Section` initializer); one `Note` per `ImportedNote` with
     `id = NoteIdentity.uuid(forAnkiGUID:)`, the mapped content fields, `isDeleted: false`, and
     its section; one `CardSchedule` per note in state `.new` (`stepIndex 0`,
     `intervalDays 0`, `lapses 0`, `dueAt nil`, `easeFactor 2.5` — or the model's own defaults
     if its initializer provides them). Save the context; return the deck's `id`.

`Tests/UnitTests/ApkgImporterTests.swift` (XCTest) passes. Setup: in-memory
`ModelContainer(for: …, configurations: ModelConfiguration(isStoredInMemoryOnly: true))`, a temp
media base directory, `ApkgImporter(container:mediaBaseURL:)`, and
`try await importer.importDeck(from: Fixture.kanaDeckURL)` (helper from
`Tests/UnitTests/FixtureLocator.swift`). Assert:

- Exactly 1 `Deck` exists; `sourceDeckName == name` and both are non-empty.
- Exactly 114 `Note` rows exist, all with `isDeleted == false`.
- EVERY note has a non-nil `schedule` with `state == .new`, and non-empty `units`.
- `NoteIdentity.uuid(forAnkiGUID: "abc")` equals itself on a second call and differs from
  `NoteIdentity.uuid(forAnkiGUID: "abd")`.
- Importing a temp file of garbage bytes throws `ImporterError.badZip`.
